# 常数空间 - 使用说明

## 项目简介
常数空间是一款专注于轻量高效的UI输入工具平台，为用户提供简洁、流畅的输入体验。

## 快速开始

### 方法一：使用Python HTTP服务器（推荐，无需安装额外依赖）

```bash
# 启动服务器并绑定到0.0.0.0地址
python -m http.server 8080 --bind 0.0.0.0
```

服务器将在 http://localhost:8080 上运行，同时绑定到 0.0.0.0 地址以允许网络访问。

### 方法二：使用Node.js服务器

#### 1. 安装依赖
```bash
npm install
```

#### 2. 启动服务器
```bash
npm start
```

服务器将在 http://localhost:3001 上运行，同时绑定到 0.0.0.0 地址以允许网络访问。

## 手机访问配置

### 前提条件
- 电脑和手机连接到同一个WiFi网络
- 服务器已启动

### 步骤1：查看电脑的本地IP地址

#### Windows系统
```bash
ipconfig
```

在输出中找到 "无线局域网适配器 WLAN" 部分，查看 "IPv4 地址"，例如：`192.168.1.100`

#### macOS系统
```bash
ifconfig | grep inet
```

找到以 `192.168.` 开头的IP地址

### 步骤2：放行端口（Windows系统）

如果Windows防火墙阻止了连接，需要放行8080端口：

```bash
# 以管理员身份运行命令提示符
netsh advfirewall firewall add rule name="Constant Space Server" dir=in action=allow protocol=TCP localport=8080
```

### 步骤3：手机访问

在手机浏览器中输入：
```
http://[电脑IP地址]:8080/未命名.html
```

例如：
```
http://192.168.1.100:8080/未命名.html
```

## 功能说明

- **搜索功能**：输入关键词后点击搜索按钮，系统会调用后端API获取搜索结果
- **多语言支持**：支持中文和英文切换
- **隐私保护**：严格遵守隐私保护规范，所有输入内容仅在本地处理

## 项目结构

- `未命名.html` - 前端主页面
- `server.js` - 后端服务器
- `package.json` - 项目配置和依赖

## 技术栈

- **前端**：HTML5, CSS3, JavaScript, Tailwind CSS, Font Awesome
- **后端**：Node.js, Express

## 注意事项

- 服务器默认运行在3001端口
- 确保电脑和手机在同一WiFi网络下
- 如需修改端口，可编辑 `server.js` 文件中的 `PORT` 常量

## 故障排查

### 手机无法访问
1. 检查电脑和手机是否在同一WiFi网络
2. 确认服务器是否正常运行（查看命令行输出）
3. 检查防火墙是否阻止了连接
4. 尝试在电脑浏览器中使用本地IP地址访问，确认服务器可通过网络访问

### 搜索功能无响应
1. 确认服务器已启动
2. 检查浏览器控制台是否有错误信息
3. 查看命令行输出，确认服务器是否收到搜索请求
